# Camping Meetup Finder

서울, 광명, 대전, 하남 출발자가 함께 모이기 좋은 캠핑장을 이동시간 중심으로 비교하는 정적 웹앱입니다.

## 기능

- 출발지별 예상 이동시간 비교
- 이동시간 균형, 최장 이동시간, 예약 가능 우선 정렬
- 예약 가능 확인 후보 필터
- 공식 예약 페이지와 지도 검색 링크 연결

## 로컬 실행

정적 파일만 사용하므로 `index.html`을 브라우저에서 열면 됩니다.

```bash
npx serve .
```

## 배포

GitHub Pages에서 루트 디렉터리를 배포 대상으로 지정하면 바로 동작합니다.
